什么是 Reto

阿里公司开发的基于 hooks 的 React Store，小巧灵活并且高效。
这里是地址。

这是一个基于 React Hooks 开发的，所以 React 的版本只能在16.7以上。

特性

  • 支持全部的 React Hooks
  • 简单高效，容易上手（确实）
  • 告别单一树，可定义多个 store，随用随取
  • 基于 React Context 的依赖注入
  • 支持 TS，兼容 JS
