创建博客

发表于 2019-10-11 更新于 2021-05-29

这个网站使用的工具是Hexo
建站的方法参考该文
本站使用的主题为NexT

这个网站的由来

也不算心血来潮，也没有预谋很久，反正我现在需要一个这样的网站，工作三个月以来一直使用石墨文档记录自己每天的工作，很琐碎也很庞杂。可能是最近比较闲（打死），所以花了半个下午搞这么一个网站

先行知识

使用hexo来创建自己的博客时，需要具有一定的先行知识

  • Git: 能够基础clone和push
  • Node: 稍微熟悉npm包管理的命令行操作
  • Markdown: 需要使用该语法来写文档

基本创建步骤

npm install -g hexo-cli // 使用npm下载hexo搭建工具

hexo init folder && cd name // 在合适的地方创建该项目并进入

hexo new name // 创建文章

hexo generate | hexo g // 生成网站

hexo serve | hexo s // 本地预览

hexo delay | hexo d // 发布至github

创建该网站遇到的问题

  1. 时区timezone设置错误报错: Cannot read property 'utcOffset' of null

    • 错误原因

      我设置的时区为CN（错误设置时区）

    • 解决方法

      中国的时区应该是：Asia/Shanghai，修改就ok。

  2. 更换主题时，从GitHub直接clone主题至根目录，修改站点配置之后，运行hexo g出现warn

    • 错误原因

      将主题错误下载至根目录，并且新主题的文件夹名称和主题不一致

    • 解决方法

      将下载的目录移至theme目录，并修改名称与主题名称一致

      或者 下载时直接clone至对应文件夹如 git clone https://github.com/theme-next/hexo-theme-next.git theme/next

  3. 根据网上许多人经验添加网易云背景音乐，网上的方法都是在next/layout/_macro/sidebar.swig中加，并没有效果

    • 解决方法

      在next\layout\_partials\sidebar\site-overview.swig这个文件中添加

  4. 如何添加标签页面

    a. 首先运行hexo new page tags

    b. 在生成的source/tags/index.md添加type: "tags"

    c. 然后在自己文章的中添加tags就ok啦。

  5. 首页文章太长如何只显示预览

    a. 找到主题的配置文件next/_config.yml

    b. 将auto_excerpt的enable修改为true即可

    这种方法在本文中无法使用

    hexo官方推荐在文章中插入<!-- more -->，可以自动显示预览

  6. hexo d发布时出现错误hexo g 报错 The "mode" argument must be integer.

    a. 出现这个错误时，说明本地Node版本过高，导致hexo发布错误。

    b. 可以使用nvm来对Node版本进行管理
