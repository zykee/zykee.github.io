该应用实现以下功能

  1. 用户简单登录以及登录限制
  2. 用户登录提醒
  3. 发送、接受信息
  4. 保存部分聊天记录
  5. 显示正在输入

socket.io

socket.io是一个Websocket库，支持及时双向的基于事件的交流

它有以下特点：

  • 实时分析：将数据推送到客户端，这些客户端会被表示为实时计数器，图表或日志客户
  • 实时通信与聊天：简单方便，该应用便是基于此特点
  • 二进制流传输：支持任何二进制文件传输，例如图片，视频，音频等
  • 文档合并：允许多个用户户同时对文档进行编辑，并且能看到每个用户进行的修改

Javascript 作为一门跑单线程的语言，必然会在许多方面受到限制，所以异步的概念就就此火了起来。

什么是异步

异步首先是相对于同步而言，就是非同步。同步执行的程序它是顺序执行的，就是连续的执行多个任务。所以异步的程序可以理解为不连续的执行。

当你在点击一个下载按钮的时候，下载任务开始，但是下载的任务是漫长的，这个时候你就可以异步的进行下载，然后去干其他你想干的事情。
